SSPak

SSPak is a database and asset bundling tool for SilverStripe.

Installation: Linux and OSX

Method 1: Install script

Use the install script, where the final argument is the directory where the binary will be installed to.

curl -sS http://silverstripe.github.io/sspak/install | php -- /usr/local/bin

If the final argument is omitted, the script will be installed into the current directory. If you don't have permission to write to the directory, "sudo" will be used to escalate permissions.

Method 2: Direct download

If you prefer not to use the installer, you can download the script and copy it to your executable path:

chmod +x sspak.phar
sudo mv sspak.phar /usr/local/bin/sspak

Installation: Windows

We assume here the program will be installed to "c:\bin" and that your path environment variable is set up to point to this directory.

On Windows 8 the path can be set by: pressing Windows-SysRq, clicking "Advanced system settings", then "Environment Variables...", then modifying the "Path" entry under "System variables". Refer to Google for instructions for your specific system.

Method 1: msysgit bash

Open your git bash and double check your PATH variable contains both your php directory and "/c/bin":

echo $PATH

Download sspak.phar to "/C/bin" and make it executable:

cd /C/bin
php -r "readfile('http://silverstripe.github.io/sspak/sspak.phar');" > sspak
chmod a+x sspak

Note: You can also download the sspak.phar directly and move it to desired location by hand. Don't forget to rename it to "sspak" and give it execution permission.

Method 2: cmd.exe

Open your terminal and double check your Path variable contains both your php directory and "c:\bin":

echo %Path%

Download sspak.phar to "c:\bin" and make it executable by creating a bat file:

cd c:\bin
php -r "readfile('http://silverstripe.github.io/sspak/sspak.phar');" %*>sspak.phar
echo @php "%~dp0sspak.phar" %*>sspak.bat

Note: You can also download the sspak.phar directly and move it to desired location by hand. You will still need to create the bat file.

Usage

You should now be able to display sspak help by simply calling:

sspak

For further information, please visit the GitHub page.
